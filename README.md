**<p> 项目描述：<br>**
利用生成模型创建虚拟数字人，并整合语音合成技术，实现虚拟主播的直播带货与实时答疑功能<br></p><br>

**<p> 核心要点：<br>**
体验扩散模型与ControlNet进行虚拟形象生成，应用TTS技术生成语音，实现虚拟人的图文到音视频的全链路转换与交互<br></p><br>

**<p> 场景设定：<br>**
一个完整的“虚拟人直播带货+直播答疑”系统，将文生图技术生成的虚拟人形象与语音合成技术相结合，利用AI驱动技术使静态图像根据音频内容产生面部表情和口型动画，在“直播答疑”环节集成大语言模型，实现对观众提问的实时智能回答。
<br></p><br><br>


**<p> 技术难点：<br>**
1. 文生图与图像生成
2. 文本与语音合成
3. 音频驱动与动画
4. 多工具链整合
<br></p>
